This classification places Dryoathyrium henryi within the class Polypodiopsida, which is a group of vascular plants that reproduce via spores. The class Polypodiopsida is further grouped into the division Pteridophyta, which comprises all ferns. This division is part of the larger kingdom Plantae, which includes all plant species.
