classification

Dryopteris lobbii is a species of fern in the family Dryopteridaceae. This family is part of the larger group of ferns called Polypodiales. Polypodiales is a suborder within the class Polypodiopsida, which contains all ferns that reproduce via spores. The class Polypodiopsida is part of the division Pteridophyta, which consists of vascular plants that reproduce via spores. Pteridophyta is a division within the kingdom Plantae.
