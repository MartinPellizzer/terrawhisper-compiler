What herbal creams you should avoid if you have tonsillitis?

If you have tonsillitis, it's best to avoid using herbal creams that contain ingredients that can irritate your throat or make symptoms worse.

One example is Peppermint (Mentha piperita) cream, which can cause throat muscles to relax, allowing mucus to flow more easily and making it harder to swallow. Another example is Garlic (Allium sativum) cream, which can cause an allergic reaction in some people and make throat symptoms worse.

Also, avoid using creams that contain Valerian (Valeriana officinalis), which can cause drowsiness and make it harder to stay hydrated. Additionally, avoid creams with Slippery Elm (Ulmus rubra), as its soothing properties can be too intense for inflamed tonsils, potentially causing more irritation.

Finally, avoid using creams with Licorice Root (Glycyrrhiza uralensis), which can interact with medications and worsen tonsillitis symptoms.
