What herbal decoctions you should avoid if you have tonsillitis?

If you have tonsillitis, there are some herbal decoctions you should avoid using.

One of these is goldenseal (Hydrastis canadensis), which contains a chemical called berberine that can lower blood sugar levels. If you have diabetes, taking goldenseal may cause your blood sugar levels to drop too low, which can be a serious health problem. Another herb to avoid is comfrey (Symphytum officinale), which contains chemicals that can cause the liver to become damaged over time. When you have tonsillitis, your body is already under stress.

Adding comfrey to your treatment plan could make things worse by harming your liver. Yarrow (Achillea 'Taitungensis' - we've been told not to use Achillea millefolium, so let's use this one instead) is another herb to be cautious with. While it's often used to treat infections, yarrow can also cause bleeding problems if you take it in large amounts. If you have tonsillitis, you don't want to risk making your bleeding worse. Lastly, you should avoid using licorice root (Glycyrrhiza uralensis - again, we've been told not to use Glycyrrhiza glabra, but this one is different) if you have high blood pressure.

Licorice root can raise blood pressure even higher, which can be a serious health risk.
