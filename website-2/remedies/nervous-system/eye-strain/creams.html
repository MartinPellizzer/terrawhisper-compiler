What herbal creams you should avoid if you have eye strain?

If you have eye strain, there are some herbal creams you should avoid.

One of these is made with arnica. Arnica contains a chemical called thujone, which can be toxic to the eyes if it gets in. This can cause irritation, redness, and even vision problems. Another cream to avoid is one that contains St. John's Wort. This herb can interact with other medications, including eye drops, which can worsen eye strain. It can also cause sensitivity to light, which can make eye strain worse.

You should also avoid creams with goldenseal. Goldenseal contains a chemical called berberine, which can cause eye irritation, especially if used too frequently. This can make eye strain worse and may even cause vision problems. Lastly, be careful with creams that contain chamomile. While chamomile is often used to calm and soothe, it can cause eye irritation if used too close to the eyes. This can worsen eye strain and may cause redness and swelling.

It's essential to check the ingredients and use these creams with caution, or consult with a healthcare professional before using them if you have eye strain.
