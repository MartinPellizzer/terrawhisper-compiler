What herbal decoctions you should avoid if you have sore throat?

If you have a sore throat, there are some herbal decoctions you should avoid.

Slippery elm (Ulmus rubra) can be irritating if taken in large amounts or for a long time. It can cause stomach problems like nausea, vomiting, and diarrhea. Licorice root (Glycyrrhiza uralensis) has a similar problem.

It can cause high blood pressure, heart problems, and stomach issues, especially if taken in large doses. Comfrey (Symphytum officinale) contains compounds that can harm your liver and kidneys if taken in high amounts or for a long time. You should also be careful with Coltsfoot (Tussilago farfara) as it can cause stomach problems and interact with other medications.

Lastly, Elderberry (Sambucus canadensis) and Foxglove (Digitalis purpurea) are both toxic plants that can be fatal if ingested in large amounts.
