What herbal tinctures you should avoid if you have chest congestion?

If you have chest congestion, there are some herbal tinctures you should avoid.

One of them is lobelia (Lobelia inflata), which can be toxic in large amounts. Taking lobelia can slow down breathing and worsen respiratory problems. Another one to avoid is comfrey (Symphytum officinale), which contains pyrrolizidine alkaloids.

These compounds can damage liver cells and cause further health issues. Also, avoid pennyroyal (Mentha pulegium), which can cause liver damage and other problems. Black cohosh (Actaea racemosa) should also be avoided, as it can cause stomach problems and interact with other medications.

And finally, avoid coltsfoot (Tussilago farfara), which contains pyrrolizidine alkaloids like comfrey and can cause liver damage.
