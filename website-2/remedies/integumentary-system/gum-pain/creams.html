What herbal creams you should avoid if you have gum pain?

If you have gum pain, you should avoid using herbal creams that contain capsaicin, the active ingredient in hot peppers like Cayenne pepper (Capsicum annuum) and Red pepper (Capsicum frutescens).

Capsaicin can make gum pain worse by irritating the gums and making them more sensitive. It can also cause burning or stinging sensations, which can be painful for people with gum pain.

Another herbal cream to avoid is one that contains Willow bark (Salix alba), which contains salicylic acid, a compound similar to aspirin. While it may help with pain relief in some cases, it can irritate the gums and make the pain worse in others.

Additionally, herbal creams that contain Turmeric root (Curcuma longa) should be used with caution, as they can cause allergic reactions or interact with other medications in some people.
