What herbal teas you should avoid if you have acne?

If you have acne, there are some herbal teas you should avoid or use with caution.

Tea tree oil, found in some herbal teas, can be too harsh for acne-prone skin. It can cause more irritation and make your skin more sensitive. Another herbal tea to be careful with is one that contains dandelion root. Dandelion can increase urine production, which may lead to dehydration if not balanced properly. Dehydration can make acne worse.

Other herbal teas to be cautious with include those that contain burdock root and red clover. Burdock root can cause more oil production in the skin, which can clog pores and make acne worse. Red clover contains isoflavones, which can stimulate hormone production in the body. Hormones can affect skin health and lead to more acne breakouts. Lastly, be careful with herbal teas that contain goldenseal root. Goldenseal contains a compound called berberine, which can cause skin irritation and increase the risk of allergic reactions, which can worsen acne.

If you have acne, it's best to consult with a healthcare professional before trying any new herbal teas.
