What herbal decoctions you should avoid if you have red eyes?

If you have red eyes, there are some herbal decoctions you should avoid.

One of these is decoctions made from Goldenseal (Hydrastis canadensis). Goldenseal is often used to treat infections, but it can irritate the eyes and make redness worse. Another herb to avoid is Decortin (Scutellaria baicalensis). This herb is used for its anti-inflammatory properties, but it can also dry out the eyes, making redness and irritation worse. You should also avoid decoctions made from Ginkgo biloba.

While Ginkgo is good for memory and circulation, it can cause eye irritation and worsen redness. Additionally, decoctions made from Sage (Salvia officinalis) should be avoided. Sage can dry out the eyes and make redness worse, especially if used in high concentrations. Lastly, decoctions made from Eyebright's cousin, Plantain (Plantago lanceolata), should be used with caution. While Plantain can soothe eye irritation, it can also cause allergic reactions in some people, which can make red eyes worse. It's essential to note that these herbs can interact with each other or with other medications, and some people may be allergic to them.

Always talk to a healthcare professional before using any herbal decoctions, especially if you have sensitive eyes.
