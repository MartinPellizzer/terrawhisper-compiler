What herbal decoctions you should avoid if you have menstrual cramps?

If you have menstrual cramps, there are some herbal decoctions you should avoid.

One of these is decoctions made from 'Hydrastis canadensis' (goldenseal). This herb contains a chemical called berberine, which can stimulate the uterus and make cramps worse. Another herb to avoid is 'Cinchona officinalis' (cinchona), which contains quinine. Quinine can increase bleeding and worsen heavy periods.

'Rauwolfia serpentina' (Indian snakeroot) decoctions should also be avoided. This herb contains a chemical called reserpine, which can slow down the uterus and worsen cramps. 'Kava kava' (Piper methysticum) decoctions are also not recommended. Kava kava can lower blood pressure, and if it's too low, it can cause dizziness and make menstrual cramps feel worse. Finally, 'Valeriana officinalis' (valerian root) decoctions should be used with caution.

While valerian root can help with relaxation and sleep, it can also slow down the uterus and worsen cramps in some women.
