What herbal decoctions you should avoid if you have breastfeeding breast pain?

If you have breastfeeding breast pain, there are some herbal decoctions you should avoid.

One of these is pennyroyal (Mentha pulegium). Pennyroyal contains a compound called pulegone, which can be toxic to the baby if passed through breast milk. Consuming pennyroyal can cause liver damage and other health issues in the baby. Another herb to avoid is goldenseal (Hydrastis canadensis). Goldenseal contains a compound called berberine, which can stimulate the uterus and cause the baby to be born too early. This can be especially problematic if you are breastfeeding and want to avoid inducing labor.

Also, avoid consuming red clover (Trifolium pratense) if you are breastfeeding. Red clover is a source of estrogen, which can increase milk production but can also cause breast pain. However, consuming high amounts of red clover can lead to an overproduction of milk, causing breast engorgement and pain. Additionally, avoid consuming saw palmetto (Serenoa repens) if you have breastfeeding breast pain. Saw palmetto can reduce milk production, which can make breastfeeding more challenging. Lastly, avoid consuming passionflower (Passiflora incarnata) if you are breastfeeding.

Passionflower can cause drowsiness and relaxation, but it can also decrease milk production and cause breast pain.
